**Deskripsi Tugas**  

Rancang dan implementasikan halaman formulir untuk input data dan setelah diklik tombol simpan tampil outputnya, dengan menggunakan widget input sebagai berikut:
- `Text input`
- `Numeric input`
- `Date dan Time`
- `Selection`
